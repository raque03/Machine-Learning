## ✦ Enfoque del repositorio

Este repositorio está orientado al análisis cuantitativo de mercados financieros utilizando Python como herramienta principal.

Se busca entender cómo los datos financieros pueden modelarse, analizarse y utilizarse para la toma de decisiones.


---

## ✦ Desarrollo del contenido

El repositorio parte del análisis estadístico de datos financieros, donde se estudia el comportamiento de retornos y su variabilidad.

A partir de esto, se construyen modelos de portafolios que permiten analizar la relación entre riesgo y rendimiento, utilizando enfoques clásicos como Markowitz.

Posteriormente, se incorporan modelos más avanzados que capturan dinámicas del mercado, como los procesos de volatilidad y las series de tiempo.

Finalmente, se incluyen herramientas de predicción que permiten anticipar comportamientos futuros a partir de datos históricos.

---

## ✦ Núcleo de trabajo

Este repositorio se centra en:

- Construcción y análisis de portafolios  
- Optimización riesgo-retorno  
- Modelado de volatilidad  
- Series de tiempo financieras  
- Predicción de tendencias
