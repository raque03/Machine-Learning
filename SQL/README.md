## Contenido del repositorio

Este repositorio contiene ejercicios y prácticas de **SQL** enfocados en la consulta, exploración y análisis de datos dentro de bases relacionales.

El enfoque es práctico, utilizando notebooks para ejecutar consultas sobre una base de datos en SQLite.
