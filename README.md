# Machine Learning

Repositorio que organiza contenidos de inteligencia artificial bajo una estructura progresiva de aprendizaje, abarcando desde fundamentos matemáticos hasta modelos avanzados.

---

## Enfoque

El contenido está orientado a:

- Comprender los fundamentos antes de aplicar modelos
- Relacionar teoría con implementación práctica
- Desarrollar soluciones con base en estructuras matemáticas y estadísticas
- Mantener una organización clara del aprendizaje

## Organización

Cada sección corresponde a un conjunto de notebooks y materiales relacionados con un área específica.

El contenido está estructurado para permitir:

- Identificar el objetivo de cada tema
- Seguir una progresión lógica
- Revisar implementaciones prácticas en Python
